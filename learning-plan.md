# 学习计划

版本：2026-05-18 草稿

## 目标

为 AI x Web3 School 建立一个公开 proof-of-work 学习仓库：包含每日笔记、任务证据、Handbook feedback，以及连接 AI Agent workflow 与 Web3 权限、身份、支付、安全和可验证记录的小实验。

## 课程周期

- WCB 课程：AI x Web3 School。
- 状态：进行中。
- 课程日期：2026-05-18 至 2026-06-14。
- WCB metadata 显示的报名截止日期：2026-05-20。
- Week 1 主题：AI 与 Web3 基础。

## 第一周计划

### Day 1｜启动与定向

- 阅读 Handbook 首页并建立主题地图。
- 确认学员画像。
- 初始化本地学习仓库。
- 准备第一份每日打卡草稿。
- 如果出现概念不清或链接过期，启动第一条 Handbook feedback。
- WCB 任务重点：Learning Agent setup、个人 GitHub repo proof、5.18 线上活动笔记。

### Day 2｜AI 基础

- 阅读 LLM、Prompt、Context、RAG 和 Agent 章节。
- 产出：一张概念图和一个小型 prompt/context 实验。

### Day 3｜Web3 基础

- 阅读 Network、Cryptography、Wallet、Smart Contract 和 Account Abstraction 章节。
- 产出：带风险边界的钱包 / 签名 / 权限笔记。

### Day 4｜AI x Web3 桥接

- 阅读 Chain-aware Context、Web3 Tool Use 和 Agent Workflow。
- 产出：展示 human-in-the-loop 检查点的最小 workflow 设计。

### Day 5｜Agent Wallet 与 Machine Payment

- 阅读 Agent Wallet、Machine Payment、Settlement & Escrow。
- 产出：一个产品场景和 threat model。

### Day 6｜安全、隐私与验证

- 阅读 AI Security、AI Privacy、Verifiable AI、Agent Trust & Reputation。
- 产出：安全 AI x Web3 Agent 执行清单。

### Day 7｜复盘与项目方向

- 复盘每日笔记和任务证据。
- 选择一个 Hackathon 方向或一个小型公开实验。
- 从真实学习卡点中生成 Handbook feedback。

## 每日节奏

- 早上：读取 WCB Learning 页面，选择最小路径、推荐路径或挑战路径。
- 学习中：记录笔记、证据链接、问题和实验。
- 晚上：准备打卡草稿，手动提交到 WCB 或指定平台，再记录提交链接或状态。

## 个人约束

- AI 基础：有基础。
- Web3 基础：新手。
- 编程能力：能独立开发。
- 目标方向：Hackathon 项目。
- 每日时间预算：约 3 小时。
- 输出语言：中文。

## 默认 3 小时学习块

1. 45 分钟：阅读当前最弱的前置主题，目前是 Web3 基础。
2. 60 分钟：完成一个具体实践项，优先选择测试网、钱包、合约或 Agent workflow 相关任务。
3. 45 分钟：将实践转成 repo proof：笔记、截图清单、代码、图表或 README 更新。
4. 30 分钟：准备 WCB proof；如果出现学习卡点，形成一条 Handbook feedback 候选。

## 路径级别

- 最小路径：一个 Handbook 小节、一条短笔记、一份打卡草稿。
- 推荐路径：章节笔记加一个小实验或图。
- 挑战路径：将概念变成代码、demo、issue、PR 或项目设计。
