# AI Web3 School 学习日志

这是 AI x Web3 School 的个人学习日志和 proof-of-work 仓库。

- Handbook：https://aiweb3.school/zh/handbook/

- WCB 课程页面：https://web3career.build/zh/programs/AI-Web3-School

- WCB Agent API 文档：https://web3career.build/llms.txt

## 隐私规则

本仓库按可公开访问设计。不要提交 API key、钱包助记词、私钥、个人证件信息、未公开会议链接、私人联系方式或他人个人数据。

## 目录结构

- `profile.md`：学员画像和已确认偏好。
- `learning-plan.md`：当前学习计划和每周节奏。
- `daily/`：每日笔记和打卡记录。
- `tasks/`：任务笔记、证据和复盘日志。
- `experiments/`：小型 demo、原型和技术笔记。
- `handbook-feedback/`：Handbook 的问题、概念不清、链接过期和修改建议。
- `hackathon/`：项目想法、范围说明和 demo 计划。
- `submissions/`：打卡草稿和 WCB 提交记录。
- `templates/`：可复用笔记模板。

## 操作边界

Learning Agent 可以帮助阅读文档、规划任务、起草笔记和准备证据。WCB 或公开平台提交仍应由学员打开并确认后再正式提交。
