Personal learning journal and proof-of-work for AI x Web3 School.

## Privacy Rule

This repo is designed to become public. Do not commit API keys, wallet seed phrases, private keys, personal IDs, unreleased meeting links, private contact information, or other people's personal data.

## Structure

- `profile.md`: learner profile and confirmed preferences.
- `learning-plan.md`: current learning plan and weekly rhythm.
- `daily/`: daily notes and check-in records.
- `tasks/`: task notes, evidence, and review logs.
- `experiments/`: small demos, prototypes, and technical notes.
- `handbook-feedback/`: issues, unclear explanations, outdated links, and suggested improvements for the Handbook.
- `hackathon/`: project ideas, scoping notes, and demo plans.
- `submissions/`: check-in drafts and WCB submission records.
- `templates/`: reusable note templates.

## Operating Boundary

The Learning Agent can help read docs, plan tasks, draft notes, and prepare evidence. WCB or public platform submissions should still be opened and confirmed by the learner before final submission.
