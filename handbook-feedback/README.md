# Handbook Feedback

Use this folder to record feedback that can be indexed, reviewed, and possibly submitted back to the AI x Web3 School Handbook.

Good feedback is specific:

- Handbook page URL.
- Exact section or sentence if available.
- Problem type: unclear concept, outdated link, typo, missing example, structure issue, translation issue, or technical inaccuracy.
- Why it matters for learners.
- Suggested fix.
- Source: daily note, task, experiment, or discussion.

## File Naming

```text
YYYY-MM-DD-short-topic.md
```

Example:

```text
2026-05-18-agent-wallet-permission-boundary.md
```
