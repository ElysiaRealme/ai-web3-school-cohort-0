# Handbook Feedback

此目录用于记录可索引、可复盘、未来可回流给 AI x Web3 School Handbook 的反馈。

好的 feedback 应该具体：

- Handbook 页面 URL。
- 如果能定位，写出具体章节或句子。
- 问题类型：概念不清、链接过期、错别字、缺少例子、结构问题、翻译问题或技术不准确。
- 说明它为什么影响学习者。
- 给出建议改法。
- 标明来源：每日笔记、任务、实验或讨论。

## 文件命名

```text
YYYY-MM-DD-short-topic.md
```

示例：

```text
2026-05-18-agent-wallet-permission-boundary.md
```
